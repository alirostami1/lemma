# Ownership

## Responsibilities

Project owners are responsible for:

- issue triage
- PR review
- release and deployment decisions
- security report handling
- keeping architectural direction coherent

## Review Expectations

Reviews should focus on:

- correctness
- maintainability
- package boundaries
- migration safety
- tests and validation
- docs when behavior changes

## Ownership

Add package or area owners here as ownership becomes more granular.
