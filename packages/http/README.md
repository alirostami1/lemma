# @lemma/http

Shared HTTP helpers for Hono routes, validation, OpenAPI helpers, and common
response handling.

## Used By

- API app
- generated Hono route code
- bounded-context HTTP layers

## Commands

```bash
pnpm --filter @lemma/http build
pnpm --filter @lemma/http check-types
pnpm --filter @lemma/http lint
```
